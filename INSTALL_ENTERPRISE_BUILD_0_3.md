1. ZIP faylni oching.
2. Papkani `C:\xampp\htdocs\` ichiga joylashtiring yoki mavjud Build 0.2 ustiga yozing.
3. Apache va MySQL ishlayotganini tekshiring.
4. Brauzerda oching:
   `http://localhost/EduDirectionAI_Professional_Enterprise_Build_0_3/web/login.php`
5. Direktor login bilan kiring.
6. `Analytics`, `Tarix`, `Prediction natijalari`, `Profil` sahifalarini tekshiring.

Eslatma: Build 0.3 yangi jadval talab qilmaydi. Build 0.2 bazasi bilan mos.
